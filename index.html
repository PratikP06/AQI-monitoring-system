Project description

"Our IoT-based Air Quality Monitoring System provides real-time air quality index (AQI) data by collecting environmental parameters such as PM2.5, PM10, temperature, and humidity using sensors connected to an ESP module. The data is transmitted to our website, where users can view live readings, historical trends, and air quality classifications. This system helps raise awareness about pollution levels and provides actionable insights to promote healthier living environments. With features like alerts, data visualization, and recommendations, our platform ensures that users stay informed about the air they breathe." ...
